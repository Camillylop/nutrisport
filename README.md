# NutriSport

Projeto de Landing Page para Nutrição Esportiva de Alta Performance.

## Tecnologias Utilizadas

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Como rodar o projeto

1. Clone o repositório
2. Instale as dependências:
```bash
npm install
```
3. Rode o servidor de desenvolvimento:
```bash
npm run dev
```

## Funcionalidades
- Calculadora de Calorias (TMB/TDEE)
- Botão flutuante de WhatsApp
- Seções de Serviços, Depoimentos, FAQ e Contato
- Design Mobile-First
